Construction (corrected canon)

The canonical wordmark is a baked SVG — Montserrat Semi-Bold, all caps, tracked 0.1em, with the Montserrat Alternates forms supplying the A / I / Y only (the R, E, L, T stay Montserrat). You can't get this mixed-font mark from one CSS font-family, so don't re-typeset it — use the supplied vector. Recolour with fill / currentColor. The "R" lettermark (favicon) is the same R in a rounded tile — the one deliberate radius exception.
