REALITY · Year 2 Design System

Social Avatar Studies

Twenty-four circular profile marks for FB / Reddit / Instagram, all carrying the canonical REALITY wordmark (shipped SVG — never re-typeset, only recoloured or rotated). The system is all hard-cornered stamped rectangles, so the move is to treat the bubble as a riso stamp / token / coin — that reconciles a round crop with the ink-and-paper, down-shadow, "stamp don't float" DNA. Flip Day / Night and swap the category accent to see each mark re-skin live.
